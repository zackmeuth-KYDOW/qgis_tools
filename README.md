# QGIS Development Tools

Repository for Python processing scripts, QGIS graphical models, and QGIS style files.

## Usage
1. Open QGIS.
2. Go to **Settings** → **Options** → **Processing** → **Providers**.
3. Point **Scripts folders** to: `/home/zack/Documents/GitHub/qgis-tools/scripts`.
4. Point **Models folders** to: `/home/zack/Documents/GitHub/qgis-tools/models`.

Changes saved here will automatically sync across devices via `git push` and `git pull`.
